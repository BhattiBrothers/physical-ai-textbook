# Physical AI & Humanoid Robotics

Welcome to the AI-native textbook for **Physical AI & Humanoid Robotics**. This course bridges the gap between digital intelligence and physical embodiment, teaching you how to design, simulate, and deploy humanoid robots capable of natural human interactions.

## Course Overview

The future of AI extends beyond digital spaces into the physical world. This capstone quarter introduces **Physical AI**—AI systems that function in reality and comprehend physical laws. Students learn to design, simulate, and deploy humanoid robots using ROS 2, Gazebo, NVIDIA Isaac, and modern AI techniques.

### Focus and Theme
- **AI Systems in the Physical World**
- **Embodied Intelligence**
- **Bridging the gap between the digital brain and the physical body**

### Learning Outcomes
By the end of this course, you will be able to:
1. Understand Physical AI principles and embodied intelligence
2. Master ROS 2 (Robot Operating System) for robotic control
3. Simulate robots with Gazebo and Unity
4. Develop with NVIDIA Isaac AI robot platform
5. Design humanoid robots for natural interactions
6. Integrate GPT models for conversational robotics

## Course Structure

This 13-week course is divided into four modules:

### Module 1: The Robotic Nervous System (ROS 2)
- ROS 2 Nodes, Topics, and Services
- Bridging Python Agents to ROS controllers using rclpy
- Understanding URDF for humanoids

### Module 2: The Digital Twin (Gazebo & Unity)
- Physics simulation in Gazebo
- High-fidelity rendering in Unity
- Simulating sensors: LiDAR, Depth Cameras, and IMUs

### Module 3: The AI-Robot Brain (NVIDIA Isaac)
- NVIDIA Isaac Sim: Photorealistic simulation
- Isaac ROS: Hardware-accelerated VSLAM and navigation
- Nav2: Path planning for bipedal humanoid movement

### Module 4: Vision-Language-Action (VLA)
- Voice-to-Action using OpenAI Whisper
- Cognitive Planning with LLMs
- Capstone Project: The Autonomous Humanoid

## Prerequisites

### Software Background
- Basic to intermediate Python programming
- Familiarity with Linux command line (Ubuntu preferred)
- Understanding of basic AI/ML concepts

### Hardware Requirements
This course is computationally intensive. Recommended setup:
- **Workstation**: NVIDIA RTX 4070 Ti+ GPU, 64 GB RAM, Ubuntu 22.04
- **Edge Kit**: NVIDIA Jetson Orin Nano, Intel RealSense camera
- **Robot Options**: Unitree Go2 (quadruped) or Unitree G1 (humanoid)

### Cloud Alternatives
If you don't have access to high-end hardware:
- AWS/Azure cloud instances with GPU support
- NVIDIA Omniverse Cloud for simulation
- Free tiers of Qdrant, Neon Postgres for RAG chatbot development
